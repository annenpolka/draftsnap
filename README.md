# mise-draftsnap plugin

Installs the `draftsnap` CLI by downloading prebuilt binaries from GitHub Releases.

## Usage

```bash
mise plugin add draftsnap https://github.com/annenpolka/draftsnap.git#mise-plugin
mise install draftsnap@latest
mise use draftsnap@latest
```

Specify a version to pin:

```bash
mise install draftsnap@0.4.0
```
